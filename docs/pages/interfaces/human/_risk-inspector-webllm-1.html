How It Works
1.WebLLM downloads the model once into Safari's cache (~500MB–2.5GB)
2.All inference runs on your GPU via WebGPU — no network needed
3.Photos never leave your device. Zero cloud. Zero API cost.
4.If WebGPU is unavailable, fallback connects to a local network LLM
Safari Requirement
WebGPU requires Safari 17.4+ on iOS 17.4+. Enable in: Settings → Safari → Advanced → Feature Flags → WebGPU
